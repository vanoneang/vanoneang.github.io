# Module1：JavaScript Basic

  1. 解决先发出请求后接收到响应，不改变 fakeHTTPRequest 内部实现来解决这个问题，预期的打印结果应该是
It's the 3th result
It's the 3th result
It's the 3th result

const fakeHTTPRequest = (() => {
  let i = 0;
  return () => {
      i += 1
      console.log(`it will response in ${3 - i % 4}s`)
      const value = `It's the ${i}th result`
      return new Promise((r) => { setTimeout(() => r(value), (3 - i % 4) * 1000) })
  }
})();

    appKey: 'mock appKey',

  }
}
